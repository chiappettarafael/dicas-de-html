Maneiras de cria um "Hyperlinks" em HTML


Tag <button> dentro uma tag <form>:

Essa é uma forma de criar um botão que redirecionará o espectador para outra página ou link.
<form action="pagina2.html" method="get">
    <button type="submit" id="botao1"> Currículo </button>
</form>

Tag <a>:

Essa é a forma mais simples de criar um hyperlink, porém essa forma pode ser limitada e impedir a estilização.
<a href="linkdosite.com.br">
    "palavra clicável"
</a>

Tag <button>, dento da tag <a>:

Essa é uma das formas de criar um botão, ela não funciona em alguns navegadores, é uma má prática.
<a href="comary.html">
    <button> Saiba mais </button>
</a>
